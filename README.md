# SushiHub 3-Hour Coursework Starter

## What this contains
- React + Vite + Tailwind frontend
- Node + Express + MongoDB backend
- JWT authentication
- Full CRUD for sushi menu items
- Validation, protected routes, toast messages, search and professional UI
- Submission document template

## Fast setup

### Backend
```bash
cd backend
npm install
copy .env.example .env
# add your MongoDB Atlas MONGO_URI and JWT_SECRET
npm run seed
npm run dev
```

### Frontend
```bash
cd frontend
npm install
copy .env.example .env
npm run dev
```
